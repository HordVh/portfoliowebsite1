# Vova Siedykh | Portfolio Website

A modern, responsive portfolio website built with HTML5, CSS3, and JavaScript. This site showcases my professional background, technical skills, projects, and contact information.

## 🚀 Features

- **Modern Hero Section**: High-impact introduction with profile highlights.
- **Interactive Navigation**: Fixed sticky navbar with smooth scroll behavior.
- **About Me**: Professional summary and personalized bio grid.
- **Skills Grid**: Organized categorization of languages, frameworks, and tools.
- **Projects Showcase**: Detailed cards for top projects with direct links to GitHub and live demos.
- **Experience Timeline**: Vertical timeline of professional roles and achievements.
- **Contact Integration**: Multiple ways to reach out via Email, LinkedIn, and GitHub.
- **Responsive Design**: Optimized for everything from large desktops to mobile devices.

## 🛠️ Built With

- **Structure**: Semantic HTML5
- **Styling**: Custom CSS3 (with modular files like `portfolio.css`)
- **Logic**: Vanilla JavaScript & jQuery
- **Animations**: CSS-based scroll reveals and hover effects
- **Icons**: FontAwesome 5

## 💻 Local Setup

To view the website locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/HordVh/portfoliowebsite1.git
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd Portfolio-Website
    ```
3.  **Open in Browser**:
    Simply open the `index.html` file in any modern web browser.
